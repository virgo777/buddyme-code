# 天气查询技能

## 适用场景
用户提出天气查询需求时使用，例如：
- 北京今天天气如何？
- 上海明天会下雨吗？
- 广州最近一周的气温

## 执行流程
1. 从用户提问中精准提取目标城市名称
2. 查阅 `references/city-codes.md` 将城市名转换为标准拼音格式
3. 调用 `scripts/weather.py` 脚本执行天气数据查询
4. 整理查询结果，以简洁口语化格式回复用户

## 约束与异常处理
- 仅支持国内城市查询，国外城市需提示不支持
- 城市名称识别模糊时，主动向用户确认
- 网络异常、接口调用失败时，返回友好提示语
- 禁止返回未经整理的原始接口数据
